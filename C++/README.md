# Google Test
Using google test to run unit tests with cmake

## Getting Started

To run basic example make the follow steps:

1. Create a directory called build/ and enter on the new directory

´´´
$ mkdir build && cd build
´´´

2. Run cmake command

´´´
$ cmake ..
´´´

2. Excute binary file

´´´
$ ./unit_tests_basic
´´´
or

´´´
$ ./unit_tests_fixture
´´´
